# Employee Attrition Analysis using Python

## Project Overview

This project analyzes the IBM HR Employee Attrition dataset using Python. The objective is to identify factors affecting employee attrition through data analysis and visualization.

---

## Tools & Libraries

- Python
- Pandas
- Seaborn
- Matplotlib

---

## Dataset

IBM HR Analytics Employee Attrition Dataset

- Rows: 1470
- Columns: 35

---

## Project Workflow

- Load Dataset
- Data Exploration
- Missing Value Analysis
- Employee Attrition Analysis
- Department-wise Attrition
- Overtime vs Attrition
- Monthly Income vs Attrition
- Job Satisfaction vs Attrition
- Work-Life Balance vs Attrition
- Age vs Attrition
- Correlation Heatmap
- Pie Chart
- Scatter Plot

## Key Insights

- Employees working overtime showed higher attrition.
- Research & Development had the highest employee count.
- Employees who left generally had lower monthly income.
- Job Satisfaction and Work-Life Balance were associated with attrition.
- Correlation analysis revealed relationships among HR variables.
