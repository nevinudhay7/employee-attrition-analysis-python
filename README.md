# Employee Attrition Analysis using Python

## Project Overview

This project analyzes employee attrition using Python and Exploratory Data Analysis (EDA). The goal is to identify the factors that influence employee attrition and gain meaningful business insights through data visualization.

---

## Dataset

- IBM HR Analytics Employee Attrition Dataset
- Total Records: 1470
- Total Features: 35

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Project Workflow

1. Imported the dataset
2. Explored the data
3. Checked missing values
4. Performed statistical analysis
5. Created visualizations
6. Generated business insights

---

## Visualizations

The project includes the following visualizations:

- Employee Attrition Count
- Department-wise Attrition
- OverTime vs Attrition
- Monthly Income vs Attrition
- Job Satisfaction vs Attrition
- Work-Life Balance vs Attrition
- Age vs Attrition
- Correlation Heatmap
- Attrition Percentage (Pie Chart)
- Age vs Monthly Income (Scatter Plot)

---

## Key Insights

- Approximately 16% of employees left the company.
- Employees working overtime showed a higher attrition rate.
- Research & Development had the highest number of employees leaving.
- Employees with lower monthly income were more likely to leave.
- Job Satisfaction and Work-Life Balance influenced employee retention.
- Most employees stayed with the company, indicating a relatively low attrition rate.
